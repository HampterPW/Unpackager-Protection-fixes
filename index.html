Packaged Project Protector (PPP)

Prevents packaged projects from being unpackaged using Turbowarps unpackager. Please Select or drop in a HTML File

Please have a copy of source code for the project your protecting, as otherwise you might not be able to get it back.

Made by Lord cat. PPP is not affiliated with Scratch, Turbowarp, or PenguinMod. (This website was designed to look like Turbowarps unpackager.)

Supported Packagers

All scratch-like packagers should work, however PenguinMod packager has been tested the most.

Privacy

Files are processed locally on your computer and never sent to any server.
